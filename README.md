# School Development Plan
교내 개발 기획안 정리 및 개발


## 장르
- 플랫폼 게임

## 플레이 방식 
- w, a, d 또는 상, 좌, 우 키로 캐릭터를 움직일 수 있다.

  a, d 또는 좌, 우 는 캐릭터를 좌, 우로 이동시키며 w 또는 위쪽 화살표는 점프이다.

-  space bar를 눌러서 캐릭터가 보고있는 방향으로 공격할 수 있다.

## 캐릭터
- 가능하면 캐릭터는 3 종류로 궁수와 검사의 종류로 나뉜다.

### 궁수
- 체력 : 5
- 공격력 : 3
- 거리 : 10
- 공격속도 : 1sec

### 검사
- 체력 : 10
- 공격력 : 15
- 거리 : 1
- 공격속도 : 0.2sec
